# Syntecxhub_Dark_Light_Toggle
🌙 Dark/Light Mode Toggle Switch
# 📌 Project Overview
This project implements a Dark/Light Mode Toggle Switch using HTML, CSS, and JavaScript. Users can switch between dark and light themes dynamically. The selected theme is saved in Local Storage, so the preference remains even after refreshing or reopening the webpage.
# ✨ Features
- Toggle between Dark and Light modes.
- Dynamic color changes using JavaScript.
- User preference stored in Local Storage.
- Smooth transition effects between themes.
- Creative toggle switch with icons (🌙 / ☀️).
- Responsive and user-friendly design.
# 🛠️ Technologies Used
- HTML5
- CSS3
- JavaScript (ES6)
- Local Storage API
# 🚀 How It Works
1. The user clicks the toggle switch.
2. JavaScript changes the website theme.
3. The selected mode is saved in Local Storage.
4. When the page reloads, the saved theme is automatically applied.
